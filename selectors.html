CSS Селекторы

Селектор определяет, к какому элементу применять то или иное CSS-правило.

Обратите внимание - не существует селекторов, которые бы позволили выбрать родителя (содержащий контейнер) или соседа родителя или потомков соседа родителя.

Базовые Селекторы

Универсальный селектор

Выбирает все элементы. По желанию, он может быть ограничен определённым пространством имён или относиться ко всему пространству имён.
Синтаксис: * ns|* *|*
Пример: * будет соответствовать всем элементам на странице.

*
* {padding: 0; margin: 0; border: 0;}
Универсальный селектор.Глобально задает стили, будет соответствовать всем элементам на странице.

Селекторы по типу элемента

Этот базовый селектор выбирает тип элементов, к которым будет применяться правило.
Синтаксис: элемент
Пример: селектор div выберет все элементы<div>.

element
div {...}
Для всех элементов < div >
element
p {...}
Для всех элементов <p>

Селекторы по классу

Этот базовый селектор выбирает тип элементов, к которым будет применяться правило. Синтаксис: .имяКласса Пример: селектор .wrapper выберет все элементы с соответствующим классом (который был определён в атрибуте class="wrapper").

.class="wrapper"
.wrapper {...}
Элемент с классом "name"

Селекторы по идентификатору

Этот базовый селектор выбирает элементы, основываясь на значении их id атрибута. Не забывайте, что идентификатор должен быть уникальным, т. е. использоваться только для одного элемента в HTML-документе.
Синтаксис: #имяИдентификатора
Пример: селектор #name выберет элемент с идентификатором name (который был определён в атрибуте id="name").

#id="name"
#name {...}
Селекторы по типу элементаan>"name"

Селекторы по атрибуту

Этот селектор выбирает все элементы, имеющие данный атрибут или атрибут с определённым значением.
Синтаксис: [attr] [attr=value] [attr~=value] [attr|=value] [attr^=value] [attr$=value] [attr*=value]
Пример: селектор [autoplay] выберет все элементы, у которых есть атрибут autoplay (независимо от его значения).
Ещё пример: a[href$=".jpg"] выберет все ссылки, у которых адрес заканчивается на ".jpg".
Ещё пример: a[href^="https"] выберет все ссылки, у которых адрес начинается на "https".

[атрибут]
[target]
Все элементы с атрибутом target
[атрибут=значение]
[target=_blank]
Все элементы с target="_blank"
[атрибут~=значение]
[title~=flower]
Все элементы с атрибутом title, содержащие слово "flower"
[атрибут|=значение]
[lang|=en]
Все элементы со значением атрибута lang, начиная с "en"
[атрибут^=значение]
a[href^="https"]
Выбирает каждый элемент <a>, значение атрибута href который начинается с "https"
[attribute$=значение]
a[href$=".pdf"]
Выбирает каждый элемент <a>, значение атрибута href который заканчивается на ".pdf"
[атрибут*=значение]
a[href*="schoolsw3"]
Выбирает каждый элемент <a>, значение атрибута href которого содержит подстроку "schoolsw3"

Комбинаторы

Комбинатор запятая

Комбинатор , это способ группировки, он выбирает все совпадающие узлы.
Синтаксис: A, B
Пример: div, span выберет оба элемента - и <div> и <span>.

element,element
div, span {...}
Для всех элементов <div> и всех элементов class="red"><span>

Комбинатор потомков

Комбинатор ' ' (пробел) выбирает элементы, которые находятся внутри указанного элемента (вне зависимости от уровня вложенности).
Синтаксис: A B
Пример: селектор div span выберет все элементы <span>, которые находятся внутри элемента <div>.

element element
div span {...}
Для всех элементов <span> внутри элементов <div>

Дочерние селекторы

Комбинатор '>' в отличие от пробела выбирает только те элементы, которые являются дочерними непосредственно по отношению к указанному элементу.
Синтаксис: A > B Пример: селектор ul > li выберет только дочерние элементы <li>, которые находятся внутри, на первом уровне вложенности по отношению к элементу <ul>.

element>element
div > p {..}
Для всех элементов <p>, родителем которых является элемент <div>

Комбинатор всех соседних элементов

Комбинатор '~' выбирает элементы, которые находятся на этом же уровне вложенности, после указанного элемента, с тем же родителем.
Синтаксис: A ~ B Пример: p ~ span выберет все элементы <span>, которые находятся после элемента <p>; внутри одного родителя.

element~element
div~p
Для всех (каждого элемента) элементов <p> расположенных после элементов <div>

Комбинатор следующего соседнего элемента

Комбинатор '+' выбирает элемент, который находится непосредственно после указанного элемента, если у них общий родитель.
Синтаксис: A + B
Пример: селектор ul + li выберет любой <li>; элемент, который находится непосредственно после элемента <ul>.

element+element
div+p {...}
Для элемента <p> который размещается первым после элементов <div>

Псевдоклассы

Знак : позволяет выбрать элементы, основываясь на информации, которой нет в дереве элементов.
Пример: a:visited соответствует всем элементам <a> которые имеют статус "посещённые".
Ещё пример: div:hover соответствует элементу, над которым проходит указатель мыши.
Ещё пример: input:focus соответствует полю ввода, которое получило фокус.

Псевдоэлементы

Знак :: позволяет выбрать вещи, которых нет в HTML. Пример: p::first-line соответствует первой линии абзаца <p>.

::after
p::after
Вставляет что-нибудь после содержимого каждого элемента <p>
::before
p::before
Вставить что-то перед содержимым каждого элемента <p>
::first-line
p::first-line
Выбирает первую строку каждого элемента <p>
::first-letter
p::first-letter
Выбирает первую букву каждого элемента <p>
:first-of-type
p:first-of-type
Выбирает каждый элемент <p>, который является первым элементом <p> своего родителя
::selection
::selection
Выделяет часть элемента, выбранную пользователем
